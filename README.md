The repository holds the code and data for the project work "False Data Injection Attack Detection on Cyber Physical Power System using Machine Learning Methods" done under the guidance of Professor Kalyan Chatterjee in the department if Electrical Engineering In IIT(ISM) Dhanbad.
Following is the description of the files in the repository:-
Admittance Matrix :- CSV File holding the information about an standard IEEE 118 bus system. 
FDIA_Detection_Using_Semi_Supervised :- Code for Semi Supervised Learning Algorithms for FDIA detection.
FDIA_Detection_Using_CNN :- Code for a convolutional Neural Network to detect FDIA.
FDIA_Detection_Using_GNN :- Code for a data handeling and design of a Graph Neural Netwrok for FDIA detection.
